# 🖼️ 资源文件库 (assets)

## 🎯 文件夹用途
统一管理项目过程中的非文本资源，如图片、图表源文件、产品原型等。方便在 Markdown 中进行相对路径引用。

## 📁 子文件夹说明
- `images/`：普通的参考配图、截图和切图素材。
- `icons/`：SVG、PNG 等小图标资源。
- `mockups/`：线框图截图、高保真设计稿。
- `diagrams/`：架构图、流程图源文件（支持保存 `.drawio`, `.mermaid` 或源文件）。

## 📝 资源命名规范
建议格式：`[模块名]-[内容描述]-[版本号].扩展名`
例如：`login-flow-v1.png`

## 🔗 引用方式
在 PRD 中引用图片，推荐使用相对路径：
`![登录流程图](../assets/images/login-flow-v1.png)`
