# 📁 产品经理 AI IDE 工作空间

## 📌 项目简介
本项目工作空间专为产品经理设计，用于协同 AI（如 Claude/Gemini）高效完成需求分析、PRD 撰写、逻辑梳理和业务分析等日常工作。

## 🌳 项目结构
```text
.
├── .agent/         # AI 配置中心（强制推荐：规则、技能、工作流）
├── analysis/       # 业务分析区（建议：数据分析、流程推演）
├── assets/         # 资源文件库（建议：图片、原型、流程图）
├── context/        # 项目上下文（建议：背景、目标用户）
├── docs/           # 参考文档库（建议）
├── drafts/         # 草稿区（建议：AI 初稿、临时记录）
├── outputs/        # 最终交付物（建议：对外的正式交付文档）
├── prds/           # 正式 PRD 归档（建议）
├── prompts/        # 提示词库（建议）
├── templates/      # 模板库（建议）
└── AGENT.md        # 全局知识库（强制推荐：供 AI 学习当前项目）
```

## 🚀 快速开始
1. 完善 `AGENT.md` 中的项目背景，帮助 AI 建立基础认知。
2. 查看 `.agent/workflows/` 了解当前的标准 AI 工作流。
3. 在 `drafts/` 新建草稿文档，通过对话让 AI 辅助你撰写内容。
4. 将确认定稿的 PRD 转移到 `prds/` 或 `outputs/` 目录。

## 💡 使用技巧
- **保持上下文同步**：经常更新 `AGENT.md` 和 `context/` 目录，AI 的回答质量取决于它获取到的上下文。
- **拆分任务**：复杂的 PRD 建议先在 `analysis/` 进行逻辑拆解，再由 AI 生成最终规范。
