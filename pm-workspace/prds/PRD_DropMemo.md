## 一、产品概述

### 1.1 产品定位

DropMemo 是一款专为 MacBook 刘海屏用户设计的轻量级便签工具。鼠标悬停刘海区域即展开便签面板，移开即收起——让便签触手可及，同时不占用任何桌面空间。

### 1.2 核心价值

| 价值点 | 描述 |
|---|---|
| **零打断** | 不需要切换窗口，刘海区悬停即用 |
| **多端同步** | iCloud 自动同步，多台 Mac 均可访问 |
| **轻量高效** | 支持待办、加粗、颜色等实用格式，够用不臃肿 |
| **随时导出** | 一键导出 Markdown，数据永远属于自己 |

### 1.3 目标用户

- MacBook 刘海屏用户（M1 Pro/M2/M3/M4 系列）
- 日常需要快速记录想法、待办事项的知识工作者
- 偏好 Markdown 格式管理笔记的用户

---

## 二、产品目标

### V1.0 目标

交付一个可日常使用的便签工具，核心体验完整：悬浮触发 + 多标签 + 富文本 + iCloud 同步 + Markdown 导出。

### 成功指标

- 悬浮触发响应时间 < 100ms
- 动画帧率稳定 ≥ 60fps
- 数据无丢失（断网也可正常使用，联网自动同步）
- 冷启动后台时间 < 2 秒

## 三、功能详细需求

### 3.1 触发机制 — 刘海悬浮触发

**交互流程：**

```
鼠标移入刘海热区
  └─→ 延迟 150ms（防误触）
      └─→ 面板从上向下滑出（动画 250ms，ease-out）
          └─→ 鼠标离开面板 & 热区
              └─→ 延迟 400ms（防抖）
                  └─→ 面板向上收起（动画 200ms，ease-in）
```

**技术实现：**

- 在刘海位置创建一个 **透明热区窗口**（宽 200px，高 40px，完全透明）
- 轮询鼠标位置（~60fps），判断是否在热区内
- 面板作为独立 **无边框浮动窗口**（始终置顶）

**边缘情况处理：**

- 鼠标在面板内活动：不触发收起
- 鼠标从热区直接滑入面板：不触发收起
- 全屏模式：不处理（用户确认全屏时不需要便签）
- 外接显示器：仅在主显示器（含刘海的那台）生效

### 3.2 面板 UI

**面板规格：**

| 属性 | 值 |
|---|---|
| 宽度 | 480px（固定） |
| 高度 | 560px（固定） |
| 圆角 | 0px 顶部（紧贴刘海），16px 底部 |
| 背景 | 毛玻璃效果 + 自定义颜色叠加 |
| 位置 | 屏幕水平居中，顶部贴合菜单栏底部 |
| 阴影 | 大阴影，增强悬浮感 |

**面板分区（上→下）：**

```
┌─────────────────────────────┐  ← 刘海
│  [标签1] [标签2] [+]    [⚙] │  ← 标签栏 (40px)
├─────────────────────────────┤
│  格式工具栏                  │  ← 工具栏 (36px)
│  [B] [I] [颜色] [☑] [—]    │
├─────────────────────────────┤
│                              │
│  文本编辑区域                │  ← 编辑器 (主区域)
│                              │
├─────────────────────────────┤
│  字数：128字  [导出 ↗]      │  ← 底部状态栏 (32px)
└─────────────────────────────┘
```

**界面语言：中文**

|---|---|---|
| 新建标签 | 点击 `+` 按钮，创建空白标签 | P0 |
| 关闭标签 | 悬停标签显示 `×`，点击关闭（有确认提示） | P0 |

**限制：**

### 3.4 富文本编辑

使用 **Tiptap 2.x**（基于 ProseMirror 的轻量富文本框架）。

#### 3.4.1 文字格式

| 格式 | 工具栏入口 | 快捷键 |
|---|---|---|
| **加粗** | `B` 按钮 | `⌘B` |
| *斜体* | `I` 按钮 | `⌘I` |
| ~~删除线~~ | `S̶` 按钮 | `⌘⇧X` |
| 字体颜色 | 调色板图标 | — |

#### 3.4.2 字体颜色选择器

- 预设 8 种常用颜色：红、橙、黄、绿、蓝、紫、粉、默认（白/黑）
- 支持自定义 HEX 颜色输入
- 样式：小圆点色板，点击即选

#### 3.4.3 结构格式

| 格式 | 工具栏入口 | 快捷键 | 效果 |
|---|---|---|---|
| 无序列表 | `☰` | `⌘⇧8` | `• 项目` |
| 有序列表 | `1.` | `⌘⇧7` | `1. 项目` |
| **待办清单** | `☑` | `⌘⇧9` | `☐ 任务`（可勾选）|
| 分割线 | `—` | — | 水平分隔线 |

#### 3.4.4 待办清单特别说明

- 点击复选框切换完成/未完成状态
- 完成的条目显示删除线 + 降低不透明度
- 导出为 Markdown 时：未完成 → `- [ ]`，完成 → `- [x]`

### 3.6 数据存储 — iCloud 同步

**存储路径：**

```
~/Library/Mobile Documents/com~apple~CloudDocs/DropMemo/
  ├── notes.json          # 所有便签数据
  └── settings.json       # 用户设置（主题等）
```

**数据结构（notes.json）：**

```json
{
  "version": "1.0",
  "lastModified": "2026-05-19T14:00:00Z",
  "tabs": [
    {
      "id": "uuid-1",
      "name": "工作待办",
      "theme": "dark",
      "content": "<p>Tiptap HTML 格式内容</p>",
      "createdAt": "2026-05-19T10:00:00Z",
      "updatedAt": "2026-05-19T14:00:00Z"
    }
  ]
}
```

**同步策略：**

- 编辑后 **500ms 自动保存**到文件
- iCloud 由 macOS 系统自动同步，应用无需处理
- 离线时：正常使用本地文件，联网后自动上传

**iCloud 未启用处理：**

- 检测到 iCloud 路径不存在时，**自动降级**到本地存储 `~/Documents/DropMemo/`
- 在设置界面提示用户启用 iCloud Drive

### 3.7 导出功能

**导出为 Markdown：**

- 点击底部「导出」按钮 → 导出当前标签
- 文件名：`{标签名}_{日期}.md`
- 弹出系统文件保存对话框，默认保存到桌面

**Markdown 转换规则：**

| 富文本格式 | Markdown 输出 |
|---|---|
| **加粗** | `**文字**` |
| *斜体* | `*文字*` |
| ~~删除线~~ | `~~文字~~` |
| 字体颜色 | `<span style="color:#ff0000">文字</span>` |
| 无序列表 | `- 项目` |
| 有序列表 | `1. 项目` |
| 待办（未完成） | `- [ ] 任务` |
| 待办（已完成） | `- [x] 任务` |
| 分割线 | `---` |

## 四、交互流程

### 首次使用

```
安装应用 → 开机自动启动（后台运行）
  → 菜单栏显示 DropMemo 图标
  → 提示：「将鼠标移到刘海区域开始使用」
  → 用户悬停 → 面板展开 → 开始记录
```

### 日常使用

```
鼠标移入刘海区域
  → 面板滑出（250ms）
  → 用户编辑内容（自动保存）
  → 鼠标移开面板
  → 400ms 后面板收起（200ms）
  → 数据已保存 ✓
```

## 五、技术方案

### 5.1 技术栈

| 层级 | 技术选型 | 理由 |
|---|---|---|
| 应用框架 | **Electron 28+** | macOS API 完整支持 |
| UI 框架 | **React 18 + TypeScript** | 组件化，类型安全 |
| 富文本编辑器 | **Tiptap 2.x** | 轻量、可扩展，原生支持 Todo |
| 数据持久化 | **fs 模块**（直接写 iCloud 路径） | 系统自动同步 |
| 样式 | **Vanilla CSS** + CSS Variables | 主题切换方便 |
| 打包 | **electron-builder** | macOS DMG 打包 |

### 5.2 窗口架构

```
DropMemo 应用
  ├── 主进程 (main.js)
  │     ├── 鼠标位置轮询（60fps）
  │     ├── 面板窗口（480×560，无边框，always-on-top）
  │     ├── Tray 图标（状态栏黑白便签图标）
  │     └── IPC 通信桥
  └── 渲染进程 (React App)
        ├── 标签栏组件
        ├── 格式工具栏组件
        ├── Tiptap 编辑器
        ├── 颜色选择器
        ├── 导出模块
        └── 设置面板
```
