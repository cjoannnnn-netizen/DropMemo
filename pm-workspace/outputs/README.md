# 📦 最终交付物 (outputs)

## 🎯 文件夹用途
存放经过反复打磨、最终可以对外发出的正式交付物。

## 📁 子文件夹说明
- `client-prds/`：给客户或业务方看的正式 PRD（通常排版精美、已脱敏）。
- `presentations/`：汇报演示材料、Slide 脚本、大纲。
- `handoff-docs/`：给研发、测试的设计交接文档。
- `archive/`：历史交付物的归档。

## 📝 文件命名规范
建议包含完整的版本和日期：
`[项目名]_[文档类型]_[版本号]_[发布日期].扩展名`

## ✨ 质量标准
放入本目录的文档应该：
1. 逻辑自洽，无未解决的问题（TBD）。
2. 排版整洁，符合团队规范。
3. 经过最终的人工走查。
